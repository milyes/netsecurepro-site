DZROUGE TERMINAL IA

> Interface CLI réactive propulsée par une IA locale (offline) et connectée (online) via NetSecurePro






---

🔌 Description

DZROUGE TERMINAL IA est une interface intelligente animée qui simule un terminal IA réactif rouge, avec des fonctions locales et distantes. Il intègre :

Un moteur IA offline simulé (chatgpt.bin)

Une interface HTML embarquée (dzrouge.html / index.html)

Un menu intelligent (dz_menu.sh)

Des modules de réponse vocale, lecture de fichiers .bin et intégration à NetSecurePro


> Conçu pour fonctionner sur Termux (Android) et sur environnement Linux CLI.




---

⚙️ Installation (Termux / Linux)

git clone https://github.com/milyes/DZROUGE_TERMINAL_IA.git
cd DZROUGE_TERMINAL_IA
bash dz_terminal_ai_fullstarter.sh

Lancez ensuite le terminal avec :

./dzterminal
