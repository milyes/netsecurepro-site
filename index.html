🛠️ NetSecurePro - Auto Assign GitHub Workflow

Ce module GitHub permet l'attribution automatique des tickets (issues) et des pull requests.

📄 Fichier auto-assign.yml


nom : Attribution automatique

sur :
  problèmes :
    types : [ouvert]
  pull_request :
    types : [ouvert]

emplois :
  courir :
    fonctionne sur : ubuntu-latest
    autorisations :
      problèmes : écrire
      pull-requests : écriture
    étapes :
      - nom : « Problème d'attribution automatique »
        utilisations : pozil/auto-assign-issue@v1
        avec :
          jeton de dépôt : ${{ secrets.GITHUB_TOKEN }}
          cessionnaires : [milyes, netsecurebot]
          nombre de cessionnaires : 1

✅ Intégration

  • Placez ce fichier dans .github/workflows/ de votre dépôt GitHub.
  • Activez GitHub Pages via Settings > Pages sur la branche main dans le dossier /docs.
